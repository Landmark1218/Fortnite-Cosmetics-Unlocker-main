# Fortnite-Cosmetics-Unlocker-For-PIE

Credit:BiruFN,Landmark0920

When using the exchange code to log into PIE, you can unlock all Cosmetics.

# How to use

Please replace the DLL located in
Fortnite\FortniteGame\Binaries\Win64
with the one provided.
You can get it from https://discord.gg/aKpUw6EYxD
After that, simply launch the tool — that’s all you need to do.

# Note:
1.Please do not close this tool using the X button in the upper right corner. We recommend that you always press a key within the window to close it.
If you accidentally close this tool using the X button in the top right, please manually turn off the proxy settings.

2.This tool might be flagged as a virus by Windows Defender or other antivirus software,
so make sure to add the folder to the exclusion list in Windows Defender
(or do the equivalent in your security app).
